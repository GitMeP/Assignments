# GenAI Assignments

This repository contains **GenAI assignments** implemented using Python and Jupyter Notebooks.

## Files Included
- `Assignment1_KNN_Loan_Default.ipynb` – Loan default prediction using KNN  
- `Assignment2_Loan_Default_Interpretation.ipynb` – Theory and risk analysis  
- `Assignment2_Neural_Network_Visualization.ipynb` – Neural network architecture for fraud detection  
- `Assignment3_3D_Regression_House_Price.ipynb` – 3D regression visualization  
  

## Tools Used
Python, Jupyter Notebook, Pandas, NumPy, Matplotlib, Scikit-learn

## How to Run
Open the notebooks in Jupyter and run all cells.
